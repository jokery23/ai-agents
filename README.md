# AI Agent

A collection of rules and skills, compatible with multiple AI coding assistants.

## Setup

Copy folders `rules` and `skills` to `~/.<assistant_name>/` directory.

or use symlinks:

### Claude Code

```bash
ln -s ~/ai-agents/rules ~/.claude/rules
ln -s ~/ai-agents/skills ~/.claude/skills
```

### Gemini CLI

```bash
ln -s ~/ai-agents/rules ~/.gemini/rules
ln -s ~/ai-agents/skills ~/.gemini/skills
```
