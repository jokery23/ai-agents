# AI Agent

A collection of rules and skills for Angular and NestJS development, compatible with multiple AI coding assistants.

## Setup

### Claude Code

```bash
ln -s ~/ai-agent/rules ~/.claude/rules
ln -s ~/ai-agent/skills ~/.claude/skills
```

> Make sure `~/.claude/` directory exists first.

### Cursor

Cursor supports `.mdc` rule files natively via the `~/.cursor/rules/` directory:

```bash
ln -s ~/ai-agent/rules ~/.cursor/rules
ln -s ~/ai-agent/skills ~/.cursor/skills
```

### Windsurf (Codeium)

```bash
ln -s ~/ai-agent/rules ~/.windsurf/rules
ln -s ~/ai-agent/skills ~/.windsurf/skills
```

### Gemini CLI

```bash
ln -s ~/ai-agent/rules ~/.gemini/rules
ln -s ~/ai-agent/skills ~/.gemini/skills
```

### OpenAI Codex CLI

```bash
ln -s ~/ai-agent/rules ~/.codex/rules
ln -s ~/ai-agent/skills ~/.codex/skills
```
